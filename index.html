bar chart example

Create chart

// create Chart.js chart from canvas element context and bar chart options let ctx = document.getElementById("chart").getContext("2d"); let chart = new Chart(ctx, coffeeChart);

Add alt format, no toggle

The alternative format for your chart stays visually hidden by default. The alt format is still always accessible to screen readers.

// add chart alternative with overview addChartAlt(chart);

Add alt format, toggleable

Set toggle true to allow users to switch between viewing the chart and its alt format.

// add chart alternative with toggle and overview addChartAlt(chart, true);
